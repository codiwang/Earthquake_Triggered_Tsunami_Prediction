# Earthquake_Triggered_Tsunami_Prediction
Binary classification project for predicting whether an earthquake event is associated with a tsunami using machine learning models.

### Environment & Dependencies
- Python 3
- NumPy
- pandas
- matplotlib
- seaborn
- scikit-learn

Install dependencies with:
`pip install numpy pandas matplotlib seaborn scikit-learn`

### How to Run
1. Open Final_Project_Code.ipynb

2. Upload dataset file earthquake_data_tsunami.csv to the same directory

3. Run all cells sequentially

4. Review printed metrics and generated plots for model comparison
